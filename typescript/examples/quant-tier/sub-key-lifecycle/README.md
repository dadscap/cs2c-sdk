# Sub-Key Lifecycle

Demonstrates `GET`, `POST`, `PATCH`, reissue, and `DELETE` operations under `/v1/account/sub-keys`.

```bash
npm run example -- examples/quant-tier/sub-key-lifecycle/sub_key_lifecycle.ts
npm run example -- examples/quant-tier/sub-key-lifecycle/sub_key_lifecycle.ts --keep-resources
```
