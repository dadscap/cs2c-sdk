# Quant Signals and Arbitrage

Review technical indicators for one item, then print the current arbitrage leaderboard.

This script makes no writes — safe to run repeatedly.

## Tier

Quant

## Run

From the `sdk-scripts/typescript/` directory after completing setup:

```bash
npm run example -- examples/quant-tier/quant-signals-and-arbitrage/quant_signals_and_arbitrage.ts
```
