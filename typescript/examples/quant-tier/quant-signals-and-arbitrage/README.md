# Quant Signals And Arbitrage

Combines `GET /v1/market/indexes`, `GET /v1/market/arbitrage`, and `GET /v1/market/indicators` into one market-intelligence report.

```bash
npm run example -- examples/quant-tier/quant-signals-and-arbitrage/quant_signals_and_arbitrage.ts
npm run example -- examples/quant-tier/quant-signals-and-arbitrage/quant_signals_and_arbitrage.ts --min-spread-pct 2
```
