# Market Data Workbench

Builds a starter market sheet from `GET /v1/prices` and daily `GET /v1/prices/candles` data.

```bash
npm run example -- examples/free-tier/market-data-workbench/market_data_workbench.ts
npm run example -- examples/free-tier/market-data-workbench/market_data_workbench.ts --query "M4A1-S" --count 5
```
