# Pro Tier Examples

Use a pro-tier key for deeper account and analytics workflows.

- [`create-comparison-table/create_comparison_table.py`](create-comparison-table/create_comparison_table.py) - Compare price, bid, and sales data for a small item set.
- [`account-management/account_management.py`](account-management/account_management.py) - Create and clean up watchlist and alert resources.
- [`portfolio-workflow/portfolio_workflow.py`](portfolio-workflow/portfolio_workflow.py) - Create, value, and clean up a temporary portfolio.
- [`item-analytics-report/item_analytics_report.py`](item-analytics-report/item_analytics_report.py) - Review item analytics and provider coverage.

## Run

Run from the `sdk-scripts/python/` directory with your virtual environment active:

```bash
python examples/pro-tier/create-comparison-table/create_comparison_table.py
python examples/pro-tier/account-management/account_management.py
python examples/pro-tier/portfolio-workflow/portfolio_workflow.py
python examples/pro-tier/portfolio-workflow/portfolio_workflow.py --import-from-steam
python examples/pro-tier/item-analytics-report/item_analytics_report.py
```
