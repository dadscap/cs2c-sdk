# Market Data Workbench

Builds a starter market sheet from `GET /v1/prices` and daily `GET /v1/prices/candles` data.

```bash
python examples/free-tier/market-data-workbench/market_data_workbench.py
python examples/free-tier/market-data-workbench/market_data_workbench.py --query "M4A1-S" --count 5
```
