# Quant Signals And Arbitrage

Combines `GET /v1/market/indexes`, `GET /v1/market/arbitrage`, and `GET /v1/market/indicators` into one market-intelligence report.

```bash
python examples/quant-tier/quant-signals-and-arbitrage/quant_signals_and_arbitrage.py
python examples/quant-tier/quant-signals-and-arbitrage/quant_signals_and_arbitrage.py --min-spread-pct 2
```
